# Сервис бронирования комнат

## Описание
Сервис бронирования комнат позволяет пользователям создавать и управлять бронированиями комнат. Он предоставляет API для создания пользователей, комнат и бронирований, а также для проверки доступности комнат.

## Функциональность
- Создание и управление пользователями
- Создание и управление комнатами
- Создание и управление бронированиями
- Проверка доступности комнат

## Запуск приложения
Для запуска приложения используйте скрипт `start.sh`:

```bash
./start.sh
```

## Необходимые установки для локального запуска
1. **Java**: Убедитесь, что у вас установлена Java 11 или выше.
2. **SBT**: Установите SBT для сборки и запуска проекта.
3. **PostgreSQL**: Установите и настройте PostgreSQL для работы с базой данных.
4. **Docker и Docker Compose**: Установите Docker и Docker Compose для запуска приложения в контейнерах.
5. **Nginx**: Установите Nginx для маршрутизации портов.
